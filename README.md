## Servidor de desenvolvimento

Para iniciar o servidor local, execute:

```bash
npm install
npm run start
```

Depois abra o navegador em `http://localhost:4200/`.


## Build

Para compilar o projeto:

```bash
npm run build
```

Os arquivos gerados serão colocados em `dist/`.
